# Web Technology Lab – Tasks 1 to 11

This README provides an overview and description of all tasks (1–11) completed in the Web Technology Lab. Each task focuses on building fundamental skills in HTML, CSS, and JavaScript.

---

## **Task 1 – Basic HTML Structure**

**Objective:** Learn the foundation of HTML and create a simple webpage.

* Create a basic HTML document using `<!DOCTYPE html>`.
* Add headings, paragraphs, line breaks, and basic formatting tags.
* Understand `<head>` and `<body>` sections.

---

## **Task 2 – Working with Lists & Links**

**Objective:** Understand navigation and list structures.

* Create ordered and unordered lists.
* Add internal and external hyperlinks.
* Use anchor tags with attributes like `target`, `href`.

---

## **Task 3 – Tables in HTML**

**Objective:** Learn to structure data using tables.

* Create tables using `<table>`, `<tr>`, `<td>`, `<th>`.
* Add table headings, borders, and formatting.
* Understand `rowspan` and `colspan`.

---

## **Task 4 – Forms in HTML**

**Objective:** Build interactive forms.

* Use `<form>` with input fields such as text, email, password, radio, checkbox.
* Learn `name`, `value`, `placeholder`, and `required` attributes.
* Add submit and reset buttons.

---

## **Task 5 – XHTML Document**

**Objective:** Understand rules of XHTML.

* Write a valid XHTML file.
* Proper closing of all tags.
* Use lowercase tag names and attribute values in quotes.

---

## **Task 6 – CSS Styling (Internal, External, Inline)**

**Objective:** Learn how to apply CSS.

* Create an external CSS file.
* Use selectors, classes, and IDs.
* Style text, backgrounds, borders, and layout.

---

## **Task 7 – Images & Multimedia**

**Objective:** Embed media into webpages.

* Insert images using `<img>`.
* Add attributes: `src`, `alt`, `height`, `width`.
* Add videos or audio if required.

---

## **Task 8 – Layout Design with CSS**

**Objective:** Structure a webpage layout.

* Use `display`, `float`, and `position` properties.
* Create header, navigation bar, content area, sidebar, footer.
* Improve visual design using margins and padding.

---

## **Task 9 – Basic JavaScript Introduction**

**Objective:** Learn JavaScript fundamentals.

* Add JavaScript using `<script>`.
* Display output using `alert`, `console.log`, `document.write`.
* Basic variables and operators.

---

## **Task 10 – DOM Manipulation**

**Objective:** Work with the Document Object Model.

* Access elements using `getElementById`, `querySelector`.
* Change HTML content and styles dynamically.
* Handle events like `onclick` or `onchange`.

---

## **Task 11 – JavaScript Functions & Form Validation**

**Objective:** Use functions and validate user input.

* Create and reuse JavaScript functions.
* Validate form fields (email, password, empty input).
* Display error messages.

---

### **How to Use This Repository**

1. Open the `Tasks` folder.
2. Each task has its own subfolder (`Task_1`, `Task_2`, ..., `Task_11`).
3. View the HTML/CSS/JS files inside each task.
4. Open HTML files in the browser to test.
