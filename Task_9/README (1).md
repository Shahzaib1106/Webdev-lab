# To-Do List App (Lab 9)

## Features

-   Add new tasks
-   Edit tasks
-   Delete tasks
-   Mark tasks as completed
-   Save tasks using localStorage
-   Loads tasks automatically on refresh
-   Stylish UI and Enter key support

## How to Run

1.  Open the project in VS Code
2.  Start Live Server
3.  Use input box to add tasks
4.  Click a task to mark complete
5.  Use delete/edit buttons to update tasks
